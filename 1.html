HTML

HTML이란 무엇인가?

Hypertext Markup language (HTML) is the standard markup language for creating web pages and web applications.

In the Design menu of each page, the Backdrop feature lets you set a custom page background — whether it’s a still image; a slideshow, a video; a Google Map; a color gradient, or one of the many visual effects available. After you set a Backdrop, there is an option to “Split” the page, which displays the Backdrop and the page contents side by side.
